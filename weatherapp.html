Weather App developed with HTML, CSS & JavaScript

The process for building this project began with some research research into weather apps, leading to a simple wireframe being created using Invision Freehand. After finalising the layout and working on the UI, I began coding the skeleton of the app with HTML and Bootstap Grids. I styled it with CSS and began working on the functionality of the app.

To make the app function properly, I coded with vanilla JavaScript and had to pull live data from a weather API, which allows us to see the temperature, a short weather description, windspeed, humidity levels, and the forecast for the rest of the day - all in real time. The search bar also allows for searching any city and seeing the weather forecast for in whatever city you search.

To make the app a little more interesting, I added a function which allowed for the background image (drawn in Adobe Illustrator) to change, depending on what time of day it is. The background changes 3 times for the morning, afternoon and evening.

I'm currently in the process of building this app with React. Stay tuned!
